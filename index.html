Clique Extracted Ontologies (CliXO), Version 0.3

Author: Michael Kramer (mhkramer7@gmail.com). Trey Ideker lab, UCSD.
Release Date: 5/20/2015
CliXO release 0.3
Please cite: Kramer M, Dutkowski J, Yu M, Bafna V, Ideker T. Inferring gene ontologies from pairwise similarity data. Bioinformatics, 30: i34-i42. 2014. doi: 10.1093/bioinformatics/btu282

This code can be used to infer a directed, acyclic graph (DAG) / ontology from a set of pairwise similarity data. Please see http://bioinformatics.oxfordjournals.org/content/30/12/i34.full for a full description of the CliXO algorithm implemented here.

Please feel free to contact the author with any questions, comments, or concerns.

Code for aligning two ontologies (such as the ones created by this code) can be found here: http://mhk7.github.io/alignOntology
